# Sentiment-Analysis-using-BERT
Sentiment Analysis using BERT is a Natural Language Processing (NLP) project that analyzes text data and identifies whether the sentiment expressed is positive or negative. The project uses a pretrained BERT-based transformer model from Hugging Face to perform accurate sentiment classification on user-provided text.
The system is built using Python and the Transformers library. It utilizes the pretrained model:

distilbert-base-uncased-finetuned-sst-2-english

to predict sentiment along with a confidence score.

This project demonstrates the practical application of:

Natural Language Processing (NLP)
Transformer-based Deep Learning Models
BERT Architecture
Text Classification

The model can be used for:

Social media sentiment analysis
Product review analysis
Customer feedback systems
Opinion mining applications

The project takes text input from the user and produces sentiment predictions such as:

POSITIVE
NEGATIVE
